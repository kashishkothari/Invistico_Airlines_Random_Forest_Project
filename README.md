- Project Overview:
This project aims to predict airline passenger satisfaction using the Invistico Airline Satisfaction Dataset. Multiple ensemble learning algorithms were implemented and compared, including:

🌳 Random Forest Classifier
🚀 XGBoost Classifier
📈 Gradient Boosting Classifier
⚡ AdaBoost Classifier
The primary objective is to determine which ensemble model provides the highest classification accuracy while identifying the most influential factors affecting passenger satisfaction.

- Objectives:
Analyze the airline passenger satisfaction dataset.
Perform data preprocessing and feature engineering.
Train multiple ensemble learning models.
Compare model performance using classification metrics.
Identify the best-performing algorithm.
Analyze feature importance to understand key drivers of customer satisfaction.

Project Workflow:

1. Data Preprocessing
-Load dataset
-Handle missing values
-Encode categorical variables
-Train-test split
-Data validation

3. Exploratory Data Analysis (EDA)
-Passenger satisfaction distribution
-Customer demographics
-Correlation analysis
-Feature distributions
-Service quality analysis

4. Machine Learning Models
The following ensemble learning algorithms were implemented:

🌳 Random Forest Classifier
-Ensemble of Decision Trees
-Reduces overfitting
-Provides feature importance
-Strong baseline model
📈 Gradient Boosting Classifier
-Sequential learning approach
-Improves prediction by correcting previous errors
-High predictive performance
⚡ AdaBoost Classifier
-Adaptive boosting technique
-Focuses on misclassified observations
-Efficient for binary classification
🚀 XGBoost Classifier
-Optimized Gradient Boosting
-Regularization to prevent overfitting
-Faster and highly accurate
-Excellent performance on structured datasets

📊 Model Evaluation
Each model was evaluated using:

-Accuracy
-Precision
-Recall
-F1-Score
-Confusion Matrix
-Classification Report
